A First for Zero Trust: AWS Security Competency Status

26 Sep 2018

MAN Energy Solutions became the first organisation to achieve AWS Security Competency Status through a Zero Trust architecture — a significant milestone in the industry’s shift away from perimeter-based security.

Stelio D’Alo from Zscaler documented how MAN Energy Solutions protected its cloud infrastructure using Zscaler Private Access (ZPA), creating a model for Zero Trust in a large, complex enterprise environment.
